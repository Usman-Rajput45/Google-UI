Google UI Clone

A responsive Google Search UI clone built using HTML and Tailwind CSS. This project mimics the look and feel of Google’s homepage.

Note: This project is a Google UI clone. To view it correctly, open it in any browser other than Google Chrome with Google.com. Some browsers or direct Google search pages may prevent it from rendering properly.

Live Demo

Check the live version here: https://papaya-heliotrope-6e886e.netlify.app/

Features

Responsive design for desktop, tablet, and mobile devices.
Built with Tailwind CSS for a modern, utility-first design.
Minimalist Google-style layout with logo, search bar, and buttons.

Fully functional UI interactions (focus, hover, etc.).

Technologies Used

HTML5 – Markup structure
Tailwind CSS – Utility-first styling
Netlify – Deployment

Installation

To run this project locally:
Clone the repository:

git clone https://github.com/Usman-Rajput45/Google-UI.git


Navigate to the project folder:

cd Google-UI


Open index.html in your browser.

Usage

Type in the search bar to simulate Google search UI.

Buttons and hover effects respond like Google’s homepage.
